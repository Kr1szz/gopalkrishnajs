# gopalkrishnajs

Personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Minimalist dark design inspired by rauno.me
- Sepia light mode with theme toggle
- Mouse spotlight effect that follows cursor
- Floating dock navigation
- Mobile responsive layout
- Resume PDF link

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)
